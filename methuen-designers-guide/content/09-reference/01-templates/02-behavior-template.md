# Behavior Template

```
Behavior Name: _______________
Checks:
  - [Condition 1]
  - [Condition 2]
Action: [What to do if checks pass]
```

## Example

```
Behavior Name: Attack Nearest Enemy
Checks:
  - Is there an enemy within 10 spaces?
  - Do I have actionPoints >= 1?
Action: Move toward nearest enemy. If adjacent, perform basic attack.
```
