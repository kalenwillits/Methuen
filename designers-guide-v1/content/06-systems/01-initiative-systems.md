# Initiative Systems

## Overview

Initiative determines turn order. Campaigns must define how initiative is determined and when it's rolled.

## Default Initiative Systems

### 1. Simple Roll

**Feature**:
```
Initiative Roll (Feature)
Roll 1d20 at start of encounter
Highest result goes first each round
Re-roll ties
```

**Characteristics**:
- Random turn order
- Quick to resolve
- No stat advantage

**Best For**: Simple, fast-paced games

### 2. Resource-Modified Roll

**Feature**:
```
Dexterity Initiative (Feature)
Roll 1d10 Ã— [Dexterity] at start of encounter
Highest result goes first each round
Re-roll ties
```

**Characteristics**:
- Rewards high stats
- More consistent than pure random
- Still has variance

**Best For**: Games where stats matter

### 3. Static Initiative

**Feature**:
```
Speed-Based Initiative (Feature)
Initiative equals [Speed]
Highest Speed acts first each round
Ties broken by 1d20 roll
Fixed for entire encounter
```

**Characteristics**:
- Completely predictable
- No randomness
- Stat-based advantage

**Best For**: Strategic, planning-focused games

## Custom Initiative Design

Consider:
- How random should turn order be?
- Should stats influence initiative?
- When is initiative determined?
- Does initiative ever change mid-encounter?
